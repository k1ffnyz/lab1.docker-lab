## Краткая теоретическая часть

Docker — это платформа для разработки, доставки и запуска приложений в контейнерах.  
Контейнеры позволяют запускать приложения в изолированной среде вместе со всеми зависимостями, что обеспечивает переносимость и единообразие среды выполнения.

Основные понятия Docker:
- **Docker Image** — шаблон для создания контейнера;
- **Docker Container** — запущенный экземпляр образа;
- **Dockerfile** — файл с инструкциями для сборки образа;
- **docker-compose** — инструмент для управления несколькими контейнерами.
